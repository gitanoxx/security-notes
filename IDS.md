** Sistemas de deteccion de intrusos 
* Un IDS consiste en un conjunto de modelos y tecnicas para revelar actividades sospechosas sobre un recurso o recursos informaticos.
* Sirve para "Escuchar" todos los mensajes de entrada y salida.
* Con esto se logra deducir y/o advertir posibles ataques.

** Tipos de IDS
* Tenemos 4 tipos de IDS
- Enfoque
* Deteccion de anomalias: Analiza trafico de red para identificar lo que pueda ser normal
* Deteccion de usos: Se utiliza comparando el trafico con una base de datos
* Hibridos: Es un sistema de deteccion que combina varios IDS para ofrecer una proteccion completa.

- Origen de datos
* HIDS: Es un sistema que monitorea un servidor el cual analiza actividades sospechosas o patrones inusuales.
* NIDS: Son herramientas de seguridad que monitorean el trafico de una red en busca de actividades maliciosas.
* Hibridos: Son herramientas que recogen lo mejor de ambos tipos, permiten una deteccion local de los segmentos.

- Estructura
* Centralizados: Emplea sensores que transmiten info a un sistema central que realiza el control, esto para ahorrar.
* Distribuidos (DIDS): Es una arquitectura de seguridad que utiliza multiples sensores de deteccion de intrusiones distribuidos en diferentes puntos de una red.

- Comportamiento
* Activos: Sistema que aparte de notificar, puede ejecutar respuestas a los ataques
* Pasivos: Es un IDS que solo notifica al adm de red pero no actua
